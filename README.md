# オンデバイス駆動のAIチャットアプリ
高スペックなGPUを積んだiOSのデバイス環境で動作するローカルチャットボットです。

# MLX
Appleが開発した、機械学習ライブラリ。
Appleシリコンに最適化されたモデル形式で、Metalを活用した高速処理を実現します。
https://opensource.apple.com/projects/mlx/
